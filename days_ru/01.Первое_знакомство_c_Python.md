## История Python и эволюция языков программирования

### Происхождение Python

Python был создан в конце 1989 года Гвидо ван Россумом (Guido van Rossum) в Национальном исследовательском институте математики и информатики Нидерландов (CWI). Изначально Python планировался как преемник языка ABC, который мог бы обрабатывать системные вызовы в системе Amoeba. Гвидо хотел создать язык, который был бы простым и мощным одновременно.

Имя Python было дано в честь британской комедийной труппы "Монти Пайтон" (Monty Python), которую очень любил Гвидо. Впоследствии эмблемой Python стали две переплетенные змеи.

### Популярность Python

Python является одним из самых популярных языков программирования в мире. Согласно различным рейтингам и индексам популярности:

![TIOBE Index](res/day01/tiobe_index.png)

![GitHub, PYPL and IEEE Spectrum](res/day01/github_pypl_and_ieee_spectrum.png)

Из приведенных выше данных видно, что Python занимает лидирующие позиции в большинстве рейтингов популярности языков программирования. Его широкое применение в области науки о данных, искусственного интеллекта, веб-разработки и автоматизации делает его незаменимым инструментом для современных разработчиков.

## Установка Python-окружения

Работа готова только после подготовки инструментов. Чтобы начать свое путешествие в мир программирования на Python, сначала необходимо установить Python-окружение на компьютере, то есть установить интерпретатор Python для выполнения Python-программ. Мы рекомендуем устанавливать официальный интерпретатор Python 3, который написан на языке C и обычно называется CPython - это, вероятно, лучший выбор на данный момент. Сначала нужно найти ссылку для скачивания на [странице загрузок](https://www.python.org/downloads/) официального сайта, нажать кнопку "Download" для перехода на страницу загрузки, после чего выбрать подходящий установщик Python 3 в зависимости от своей операционной системы, как показано на рисунке ниже.

<img src="res/day01/python_download_page_1.png" style="zoom:40%;">

После входа на страницу загрузки некоторые версии Python не предоставляют установщики для систем Windows и macOS, а предоставляют только архивы с исходным кодом. Для пользователей, знакомых с системами Linux, можно установить Python, собрав его из исходного кода; для пользователей систем Windows или macOS мы **настоятельно рекомендуем** использовать установщики. Например, если вы хотите установить Python 3.10, выберите Python 3.10.10 или Python 3.10.11, чтобы найти установочные пакеты для Windows или macOS, в то время как другие версии могут предоставлять только исходный код, как показано на рисунке ниже.

5. В расширенных настройках рекомендуется отметить "Add Python to environment variables".

6. Дождитесь завершения установки.

### Установка в Linux

В большинстве дистрибутивов Linux Python уже установлен. Для проверки версии выполните:

```bash
python3 --version
```

Если Python не установлен или нужна более новая версия:

**Ubuntu/Debian:**
```bash
sudo apt update
sudo apt install python3 python3-pip
```

**CentOS/RHEL/Fedora:**
```bash
sudo yum install python3 python3-pip
# или для более новых версий Fedora:
sudo dnf install python3 python3-pip
```

### Установка в macOS

**Способ 1: Используя официальный установщик**
1. Скачайте установщик с официального сайта python.org
2. Запустите .pkg файл и следуйте инструкциям

**Способ 2: Используя Homebrew**
```bash
brew install python
```

**Способ 3: Используя macOS встроенный Python**
macOS поставляется с предустановленным Python, но это обычно старая версия. Рекомендуется установить более новую версию.

## Запуск интерпретатора Python

После успешной установки Python можно запустить интерпретатор Python несколькими способами:

### 1. Из командной строки

```bash
python
# или
python3
```

Вы увидите приглашение интерпретатора:
```python
Python 3.11.0 (main, Oct 24 2022, 18:26:48) [MSC v.1933 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

### 2. Первая программа

Введите следующий код в интерпретаторе:

```python
print("Привет, мир!")
```

Это выведет "Привет, мир!" на экран.

### 3. Интерактивный режим

В интерактивном режиме вы можете выполнять Python-код построчно:

```python
>>> 2 + 3
5
>>> name = "Python"
>>> print(f"Изучаем {name}")
Изучаем Python
```

### 4. Выход из интерпретатора

Для выхода из интерпретатора:
- Введите `exit()` или `quit()`
- Нажмите `Ctrl+Z` (Windows) или `Ctrl+D` (Linux/macOS)
