## Введение в Python

Python (британское произношение: /ˈpaɪθən/; американское произношение: /ˈpaɪθɑːn/) — это язык программирования, изобретенный голландцем Гвидо ван Россумом (Guido van Rossum). На сегодняшний день это самый популярный язык программирования в мире с наибольшим количеством пользователей. Python делает акцент на читаемости кода и простоте синтаксиса. По сравнению с такими влиятельными языками программирования, как C, C++ и Java, Python позволяет пользователям выразить свои намерения с меньшим количеством кода. ### История Python

#### Происхождение Python

1989 год: Рождественские каникулы, Амстердам. Гвидо ван Россум, чтобы скоротать время, решил разработать новый скриптовый язык — интерпретатор для языка ABC. Причины следующие:

- Увлечение языком ABC, но неудовлетворенность его недостатками
- Личный интерес к программированию и желание создать собственный язык

Название «Python» происходит не от змеи, а от британского комедийного сериала «Летающий цирк Монти Пайтона» (Monty Python's Flying Circus), который Гвидо любил смотреть.

#### Важные вехи

- 1991: Первый интерпретатор Python родился, написан на языке C (до сих пор называется CPython), способен вызывать библиотеки языка C
- 1994: Python 1.0 — включает lambda, map, filter и reduce
- 2000: Python 2.0 — включает сборку мусора, классы нового стиля
- 2008: Python 3.0 — не полностью обратно совместим, множество улучшений
- 2020: Python 2 официально прекратил поддержку (EOL)

### Преимущества Python

#### Простота и понятность

- Читаемый синтаксис, легко изучается и используется
- «Батарейки включены»: богатые стандартные библиотеки и модули
- Отличная документация и активное сообщество

#### Мощность и гибкость

- Поддержка множества парадигм: объектно-ориентированное, функциональное, процедурное программирование
- Обширная экосистема: более 400 000 пакетов в PyPI
- Кроссплатформенность: Windows, Linux, macOS

#### Широкое применение

- Веб-разработка: Django, Flask, FastAPI
- Наука о данных и машинное обучение: NumPy, Pandas, Scikit-learn, TensorFlow, PyTorch
- Автоматизация и скрипты
- Сетевое программирование, DevOps
- Игры и графика

### Недостатки Python

#### Производительность

- Интерпретируемый язык, медленнее компилируемых языков (C, C++, Java)
- GIL (Global Interpreter Lock) ограничивает многопоточность
- Решение: использование C-расширений, PyPy, Cython

#### Управление зависимостями

- Множество версий Python и пакетов может вызвать конфликты
- Решение: виртуальные окружения (venv, virtualenv, conda)

#### Мобильная разработка

- Не лучший выбор для мобильных приложений
- Есть фреймворки (Kivy, BeeWare), но не так популярны

### macOS

macOS обычно имеет предустановленный Python 2.x. Для установки Python 3:

#### Способ 1: Официальный установщик

Скачайте установщик с [python.org](https://www.python.org) и следуйте инструкциям.

#### Способ 2: Homebrew

Если у вас установлен Homebrew, выполните:

```bash
brew install python3
```

### Linux

Большинство дистрибутивов Linux имеют предустановленный Python. Для проверки версии:

```bash
python3 --version
```

Если Python 3 не установлен, используйте пакетный менеджер:

**Ubuntu/Debian:**

```bash
sudo apt update
sudo apt install python3 python3-pip
```

**Fedora:**

```bash
sudo dnf install python3 python3-pip
```

**CentOS/RHEL:**

```bash
sudo yum install python3 python3-pip
```

### Виртуальные окружения

Виртуальные окружения позволяют изолировать зависимости проектов.

#### Создание виртуального окружения

**venv (встроено в Python 3.3+):**

```bash
# Создание
python -m venv myenv

# Активация (Windows)
myenv\Scripts\activate

# Активация (macOS/Linux)
source myenv/bin/activate

# Деактивация
deactivate
```

#### virtualenv (сторонний пакет)

```bash
# Установка
pip install virtualenv

# Создание
virtualenv myenv

# Активация/деактивация — аналогично venv
```

## Первая программа на Python

### Интерактивный режим (REPL)

REPL (Read-Eval-Print Loop) — интерактивный интерпретатор Python. Запустите его, набрав `python` или `python3` в терминале:

```bash
python
```

Вы увидите приглашение `>>>`. Попробуйте:

```python
>>> print("Hello, World!")
Hello, World!
>>> 2 + 2
4
>>> exit()
```

### Запуск Python-скриптов

Создайте файл `hello.py`:

```python
print("Hello, World!")
print("Привет, Python!")
```

Запустите его:

```bash
python hello.py
```

Вывод:

```
Hello, World!
Привет, Python!
```
