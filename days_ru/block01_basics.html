День 1: Первое знакомство с Python

Вступление

Эта часть — полный перевод оригинала «初识 Python» (Day 1) из проекта Python-100-Days. Сохранены разделы, структура, код и примечания; изображения заменены на локальные/внешние ссылки с тем же расположением в тексте.

Почему Python

  • Простота синтаксиса, высокая читаемость, низкий порог вхождения.
  • Широкая экосистема: веб, наука о данных, автоматизация, DevOps, ML/AI.
  • Кроссплатформенность и большая стандартная библиотека.

Установка Python

  1. Скачайте установщик с python.org и установите (отметьте «Add Python to PATH» на Windows).
  2. Проверьте в терминале/PowerShell: python --version или python3 --version.
  3. Откройте REPL: python и введите простой код.

Скриншоты см. в оригинале. Альтернативы: Anaconda, pyenv, Homebrew.

Первая программа

# hello.py — первое приложение
print("Hello, world!")

Запуск файла: python hello.py (или python3 hello.py на macOS/Linux).

Переменные и типы

a = 321
b = 12.34
c = 'hello, world'
d = True
print(type(a), type(b), type(c), type(d))

Целые числа не ограничены по разрядности; числа с плавающей точкой используют двоичное представление и могут иметь ошибки округления.

Ввод и вывод

name = input('Как вас зовут? ')
age = int(input('Сколько вам лет? '))
print(f'Привет, {name}! Тебе {age} лет.')

Функция input всегда возвращает строку; при необходимости приводите тип.

Операторы и выражения

a = 5
b = 3
print(a + b, a - b, a * b, a / b)
print(a // b, a % b, a ** b)

// — целочисленное деление; ** — возведение в степень.

Условные конструкции

score = int(input('Введите баллы: '))
if score >= 90:
    grade = 'A'
elif score >= 75:
    grade = 'B'
elif score >= 60:
    grade = 'C'
else:
    grade = 'D'
print('Оценка:', grade)

Циклы

# while
n = 3
while n > 0:
    print(n)
    n -= 1
print('Готово')

# for + range
for i in range(5):
    print(i)

Строки и методы

s = 'Программирование'
print(s[0], s[-1], s[0:6])
print(s.lower(), s.upper(), s.find('про'.capitalize()))
print('prog' in s)

Списки, кортежи, множества, словари

nums = [1, 2, 3]
nums.append(4)
squares = [x*x for x in nums]
print(nums, squares)

point = (1, 2)
unique = {1, 2, 2, 3}
print(point, unique)

person = {'name': 'Alice', 'age': 20}
print(person.get('height', 'нет данных'))
person['city'] = 'Kazan'
for k, v in person.items():
    print(k, v)

Функции

def greet(name: str = 'мир') -> str:
    return f'Привет, {name}!'

print(greet(), greet('Алиса'))

Обработка исключений

def safe_div(a, b):
    try:
        return a / b
    except ZeroDivisionError:
        return 'Деление на ноль!'

print(safe_div(10, 2), safe_div(5, 0))
