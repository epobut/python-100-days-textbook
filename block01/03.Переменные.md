## Переменные в языке Python

Для новичков, желающих изучать программирование, есть два вопроса, на которые они, вероятно, хотят получить ответ: первый — "что такое (компьютерная) программа", второй — "что можно делать, написав (компьютерную) программу". Расскажу о своем понимании этих двух вопросов: **программа — это упорядоченная совокупность данных и инструкций**, **писать программу — значит использовать данные и инструкции для управления компьютером, чтобы он делал то, что мы от него хотим**. Сегодня так много людей выбирают язык Python для написания программ, потому что Python достаточно прост и мощен. По сравнению с такими языками программирования, как C, C++, Java, Python более дружелюбен к начинающим и непрофессионалам. Многие проблемы в языке Python можно решить простым и элегантным способом. Далее мы начнем с самых базовых элементов языка, познакомим вас с языком Python и научим его использовать.

### Некоторые основы

Перед тем как систематически начать изучение программирования на Python, давайте сначала познакомимся с базовыми знаниями о компьютерах. Аппаратная система компьютера обычно состоит из пяти основных компонентов: **арифметико-логическое устройство**, **устройство управления**, **запоминающее устройство**, **устройство ввода** и **устройство вывода**. Среди них арифметико-логическое устройство и устройство управления вместе образуют то, что мы обычно называем **центральным процессором** (CPU), его функция — выполнять различные вычисления и управляющие инструкции. Как мы упомянули ранее, программа — это совокупность инструкций, писать программу — значит организовать серию инструкций определенным образом, а затем использовать эти инструкции для управления компьютером, чтобы он делал то, что мы хотим. Запоминающее устройство можно разделить на **внутреннее запоминающее устройство** и **внешнее запоминающее устройство**. Первое — это то, что мы обычно называем оперативной памятью (RAM), это пространство хранения, к которому центральный процессор может обращаться напрямую. В процессе выполнения программы соответствующие данные и инструкции должны быть загружены в память. Устройства ввода и вывода часто называют устройствами I/O. Клавиатура, мышь, микрофон, камера — типичные устройства ввода, а монитор, принтер, динамики и т.д. — типичные устройства вывода. В настоящее время большинство компьютеров, которые мы используем, следуют "архитектуре фон Неймана". У такого компьютера есть две ключевые особенности: первая — **разделение запоминающего устройства и центрального процессора**; вторая — **кодирование данных в двоичном виде**.

Двоичная система — это метод счисления "один при переходе через два", по своей сути такой же, как используемый людьми метод счисления "один при переходе через десять". Люди используют десятичную систему счисления, потому что у них десять пальцев. При счете, когда все десять пальцев использованы, остается только использовать разряды для представления больших чисел. Конечно, всегда есть исключения. Майя, возможно, из-за того, что долгое время ходили босиком, использовали и пальцы ног, поэтому они использовали двадцатеричную систему счисления. На основе такого метода счисления календарь, используемый майя, отличался от календаря, который мы обычно используем. Согласно календарю майя, 2012 год был последним годом предыдущего так называемого "Солнечного периода", а 2013 год стал началом нового "Солнечного периода". Позже это было искажено и ошибочно передано как абсурдное утверждение, что "2012 год — это конец света, предсказанный майя". Сегодня многие предполагают, что медленное развитие цивилизации майя связано с использованием двадцатеричной системы. Для компьютера двоичную систему легче всего реализовать физически, потому что можно использовать высокое напряжение для представления 1 и низкое напряжение для представления 0. Не всем, кто пишет программы, нужно хорошо знать двоичную систему, знать преобразование между десятичной и двоичной, восьмеричной, шестнадцатеричной системами. Большую часть времени мы можем писать программы, даже не зная этих знаний. Но мы должны знать, что компьютер использует двоичную систему счисления, и какими бы ни были данные, в памяти компьютера они существуют в двоичной форме.

> **Примечание**: О двоичной системе счисления и о том, как она конвертируется в другие системы счисления, вы можете найти информацию в книгах под названием "Введение в компьютерные науки" или "Компьютерная культура", здесь мы не будем подробно останавливаться на этом. Читатели, которым это непонятно, могут изучить самостоятельно.

### Переменные и типы

Чтобы сохранить данные в памяти компьютера, сначала нужно поговорить о концепции переменной. В языках программирования **переменная — это носитель данных**. Проще говоря, это блок памяти, используемый для хранения данных. **Значение переменной можно читать и изменять**, это основа всех вычислений и управления. Компьютер может обрабатывать данные многих типов, наиболее распространенные — числовые, помимо числовых есть текст, изображения, аудио, видео и различные другие типы данных. Хотя данные в компьютере существуют в двоичной форме, мы можем использовать переменные разных типов для представления различий в типах данных. В языке Python предустановлено множество типов данных, также разрешается определять новые типы данных, об этом мы расскажем позже. Сначала давайте познакомимся с несколькими наиболее часто используемыми типами данных в Python.

1. Целочисленный тип (`int`): Python может обрабатывать целые числа любого размера и поддерживает двоичное (например, `0b100`, что в десятичной системе равно 4), восьмеричное (например, `0o100`, что в десятичной системе равно 64), десятичное (`100`) и шестнадцатеричное (например, `0x100`, что в десятичной системе равно 256) представление. Запустите код ниже и посмотрите, что будет выведено.

    ```python
    print(0b100)  # Двоичное целое число
    print(0o100)  # Восьмеричное целое число
    print(100)    # Десятичное целое число
    print(0x100)  # Шестнадцатеричное целое число
    ```

2. Тип с плавающей точкой (`float`): Числа с плавающей точкой — это десятичные дроби. Они называются числами с плавающей точкой, потому что при представлении в научной нотации позиция десятичной точки может меняться. Помимо математического написания (например, `123.456`), числа с плавающей точкой также поддерживают научную нотацию (например, `1.23456e2`, что означает $\small{1.23456 \times 10^{2}}$). Запустите код ниже и посмотрите, что будет выведено.

    ```python
    print(123.456)    # Математическое написание
    print(1.23456e2)  # Научная нотация
    ```

3. Строковый тип (`str`): Строка — это любой текст, заключенный в одинарные или двойные кавычки, например `'hello'` и `"hello"`.

4. Булев тип (`bool`): Булев тип имеет только два значения: `True` и `False`, либо `True`, либо `False`. Может использоваться для представления "да" и "нет" в реальном мире, "истина" и "ложь" утверждения, "хорошо" и "плохо" ситуации, "высокий" и "низкий" уровень и т.д. Если переменная имеет только два возможных состояния, мы можем использовать булев тип.

### Именование переменных

Для каждой переменной нам нужно дать ей имя, так же как у каждого из нас есть свое имя. В Python именование переменных должно следовать следующим правилам и соглашениям.

- Правила:
  - Правило 1: Имя переменной состоит из **букв**, **цифр** и **подчеркиваний**, цифра не может быть первой. Следует отметить, что буквы здесь означают символы Unicode. Unicode называется универсальным кодом, охватывающим большинство письменных систем мира, что означает, что китайские, японские, греческие буквы и т.д. могут использоваться в качестве символов в имени переменной, но некоторые специальные символы (например: `!`, `@`, `#` и т.д.) не могут появляться в имени переменной. Мы настоятельно рекомендуем понимать "буквы" здесь как **использование только английских букв насколько возможно**.
  - Правило 2: Python — язык программирования, **чувствительный к регистру**. Проще говоря, заглавная `A` и строчная `a` — это две разные переменные. Это, собственно, не правило, а то, на что нужно обратить внимание.
  - Правило 3: Имя переменной **не должно совпадать с ключевыми словами Python**, **старайтесь избегать зарезервированных слов Python**. Ключевые слова здесь — это слова, имеющие особое значение в программе Python (например: `is`, `if`, `else`, `for`, `while`, `True`, `False` и т.д.), зарезервированные слова в основном относятся к именам встроенных функций, встроенных модулей и т.д. языка Python (например: `int`, `print`, `input`, `str`, `math`, `os` и т.д.).
- Соглашения:
  - Соглашение 1: Имя переменной обычно использует **строчные английские буквы**, **несколько слов соединяются подчеркиванием**.
  - Соглашение 2: Защищенные переменные начинаются с одного подчеркивания.
  - Соглашение 3: Приватные переменные начинаются с двух подчеркиваний.

Соглашения 2 и 3 пока можно не учитывать, все станет понятно по ходу изучения. Конечно, как профессиональный программист, при именовании переменных важно, чтобы **имя было говорящим** — это демонстрирует профессионализм программиста, на это обращают внимание на многих собеседованиях на позиции разработчика.

### Использование переменных

Ниже на примерах объясним типы переменных и их использование.

```python
"""
Использование переменных для хранения данных и выполнения операций сложения, вычитания, умножения и деления

Version: 1.0
Author: 骆昊
"""
a = 45        # Определить переменную a, присвоить значение 45
b = 12        # Определить переменную b, присвоить значение 12
print(a, b)   # 45 12
print(a + b)  # 57
print(a - b)  # 33
print(a * b)  # 540
print(a / b)  # 3.75
```

В Python можно использовать функцию `type` для проверки типа переменной. Концепция функции в программировании очень похожа на концепцию функции в математике. Математические функции всем знакомы — они включают имя функции, независимые переменные и зависимую переменную. Если пока не понимаете концепцию функции — не страшно, мы специально объясним определение и использование функций в последующих материалах.

```python
"""
Использование функции type для проверки типа переменной

Version: 1.0
Author: 骆昊
"""
a = 100
b = 123.45
c = 'hello, world'
d = True
print(type(a))  # <class 'int'>
print(type(b))  # <class 'float'>
print(type(c))  # <class 'str'>
print(type(d))  # <class 'bool'>
```

Можно изменить тип переменной с помощью встроенных функций Python. Ниже приведены некоторые часто используемые функции, связанные с типами переменных.

- `int()`: Преобразует число или строку в целое число, можно указать систему счисления.
- `float()`: Преобразует строку (по возможности) в число с плавающей точкой.
- `str()`: Преобразует указанный объект в строковую форму, можно указать способ кодирования.
- `chr()`: Преобразует целое число (код символа) в соответствующую (односимвольную) строку.
- `ord()`: Преобразует (односимвольную) строку в соответствующее целое число (код символа).

Следующий пример демонстрирует операции преобразования типов в Python.

```python
"""
Операции преобразования типа переменной

Version: 1.0
Author: 骆昊
"""
a = 100
b = 123.45
c = '123'
d = '100'
e = '123.45'
f = 'hello, world'
g = True
print(float(a))         # int типа 100 преобразуется в float, выводит 100.0
print(int(b))           # float типа 123.45 преобразуется в int, выводит 123
print(int(c))           # str типа '123' преобразуется в int, выводит 123
print(int(c, base=16))  # str типа '123' преобразуется в int как шестнадцатеричное, выводит 291
print(int(d, base=2))   # str типа '100' преобразуется в int как двоичное, выводит 4
print(float(e))         # str типа '123.45' преобразуется в float, выводит 123.45
print(bool(f))          # str типа 'hello, world' преобразуется в bool, выводит True
print(int(g))           # bool типа True преобразуется в int, выводит 1
print(chr(a))           # int типа 100 преобразуется в str, выводит 'd'
print(ord('d'))         # str типа 'd' преобразуется в int, выводит 100
```

> **Примечание**: При преобразовании типа `str` в тип `int` можно указать систему счисления через параметр `base`, строка будет рассматриваться как целое число в соответствующей системе счисления. При преобразовании типа `str` в тип `bool`, пока строка имеет содержимое и не является `''` или `""`, соответствующее булево значение всегда `True`. При преобразовании типа `bool` в тип `int`, `True` становится `1`, `False` становится `0`. В кодировке ASCII и Unicode символ `'d'` соответствует коду `100`.

### Заключение

В программах Python мы можем **использовать переменные для хранения данных**, **переменные имеют разные типы**. Часто используемые типы — `int`, `float`, `str` и `bool`. При необходимости можно преобразовывать типы переменных с помощью встроенных функций Python. Переменные можно использовать в вычислениях, это предварительное условие для решения многих задач.
