## Операторы в языке Python

В предыдущих главах мы уже использовали различные операторы. Теперь давайте систематически изучим операторы в языке Python.

### Операторы и выражения

Мы можем использовать операторы для построения выражений, которые являются основными единицами обработки данных. Операторы выполняют определенные операции над операндами и возвращают результат.

### Операторы присваивания

В Python оператор `=` используется для присваивания значения переменной. Слева от знака равенства должно быть имя переменной, а справа - значение или выражение для присваивания переменной. Например:

```python
a = 10
b = 20
c = a + b
```

Кроме того, Python также поддерживает составные операторы присваивания, которые объединяют оператор присваивания с другими операторами. Например, `+=` означает "сложить, затем присвоить", `*=` означает "умножить, затем присвоить". Использование составных операторов присваивания может сделать код более лаконичным.

```python
a = 10
a += 5  # эквивалентно a = a + 5
print(a)  # 15
```

### Арифметические операторы

Арифметические операторы Python включают:

| Оператор | Описание | Пример | Результат |
| -------- | -------- | -------- | -------- |
| `+` | Сложение | `10 + 20` | `30` |
| `-` | Вычитание | `20 - 10` | `10` |
| `*` | Умножение | `10 * 20` | `200` |
| `/` | Деление | `20 / 10` | `2.0` |
| `//` | Целочисленное деление | `20 // 10` | `2` |
| `%` | Остаток от деления | `20 % 10` | `0` |
| `**` | Возведение в степень | `2 ** 3` | `8` |

Нужно обратить внимание:

1. Оператор `/` выполняет деление с плавающей точкой и всегда возвращает число с плавающей точкой.
2. Оператор `//` выполняет целочисленное деление и возвращает целую часть результата деления.
3. Оператор `%` используется для получения остатка от деления.
4. Оператор `**` используется для возведения в степень, например, `2 ** 3` означает 2 в степени 3.

### Операторы сравнения

Операторы сравнения используются для сравнения двух значений, результатом является логическое значение (`True` или `False`). Операторы сравнения в Python включают:

| Оператор | Описание | Пример | Результат |
| -------- | -------- | -------- | -------- |
| `==` | Равно | `10 == 10` | `True` |
| `!=` | Не равно | `10 != 10` | `False` |
| `>` | Больше | `10 > 5` | `True` |
| `<` | Меньше | `10 < 5` | `False` |
| `>=` | Больше или равно | `10 >= 10` | `True` |
| `<=` | Меньше или равно | `10 <= 5` | `False` |

Нужно обратить внимание:

1. Оператор `==` используется для сравнения, является ли значение двух операндов равным, а оператор `=` используется для присваивания.
2. Операторы сравнения можно использовать для сравнения чисел, строк и других типов данных.

### Логические операторы

Логические операторы используются для объединения нескольких условных выражений. Логические операторы в Python включают:

| Оператор | Описание | Пример | Результат |
| -------- | -------- | -------- | -------- |
| `and` | Логическое И | `True and False` | `False` |
| `or` | Логическое ИЛИ | `True or False` | `True` |
| `not` | Логическое НЕ | `not True` | `False` |

Нужно обратить внимание:

1. Оператор `and` возвращает `True`, только когда оба операнда являются `True`.
2. Оператор `or` возвращает `True`, когда хотя бы один операнд является `True`.
3. Оператор `not` используется для инверсии логического значения операнда.

### Операторы принадлежности

Операторы принадлежности используются для проверки, содержится ли элемент в последовательности (такой как строка, список, кортеж). Операторы принадлежности в Python включают:

| Оператор | Описание | Пример | Результат |
| -------- | -------- | -------- | -------- |
| `in` | Если значение найдено в последовательности, возвращает True | `3 in [1, 2, 3]` | `True` |
| `not in` | Если значение не найдено в последовательности, возвращает True | `4 not in [1, 2, 3]` | `True` |

Нужно обратить внимание:

1. Оператор `in` возвращает `True`, когда элемент найден в последовательности.
2. Оператор `not in` возвращает `True`, когда элемент не найден в последовательности.

### Операторы идентичности

Операторы идентичности используются для проверки, указывают ли две переменные на один и тот же объект в памяти. Операторы идентичности в Python включают:

| Оператор | Описание | Пример | Результат |
| -------- | -------- | -------- | -------- |
| `is` | Если две переменные указывают на один и тот же объект, возвращает True | `a is b` | `True` или `False` |
| `is not` | Если две переменные не указывают на один и тот же объект, возвращает True | `a is not b` | `True` или `False` |

Нужно обратить внимание:

1. Оператор `is` используется для проверки, являются ли два объекта одним и тем же объектом, а не для проверки равенства их значений.
2. Оператор `is` отличается от оператора `==`: `is` проверяет идентичность объектов, в то время как `==` проверяет равенство значений.

### Приоритет операторов

Приоритет операторов определяет порядок, в котором операторы выполняются в выражении. Ниже приведены некоторые распространенные операторы в порядке убывания приоритета:

1. `**` (возведение в степень)
2. `*`, `/`, `//`, `%` (умножение, деление, целочисленное деление, остаток от деления)
3. `+`, `-` (сложение, вычитание)
4. `>`, `<`, `>=`, `<=`, `==`, `!=` (операторы сравнения)
5. `not` (логическое НЕ)
6. `and` (логическое И)
7. `or` (логическое ИЛИ)

Нужно обратить внимание:

1. Для изменения порядка выполнения операторов можно использовать скобки `()`.
2. Правильное использование приоритета операторов может сделать код более ясным и читаемым.

### Практический пример

Давайте напишем простую программу для демонстрации использования различных операторов:

```python
# Арифметические операторы
a = 10
b = 20
print(a + b)  # 30
print(a - b)  # -10
print(a * b)  # 200
print(a / b)  # 0.5
print(a // b)  # 0
print(a % b)  # 10
print(a ** b)  # 100000000000000000000

# Операторы сравнения
print(a == b)  # False
print(a != b)  # True
print(a > b)  # False
print(a < b)  # True
print(a >= b)  # False
print(a <= b)  # True

# Логические операторы
print(True and False)  # False
print(True or False)  # True
print(not True)  # False

# Операторы принадлежности
my_list = [1, 2, 3, 4, 5]
print(3 in my_list)  # True
print(6 not in my_list)  # True

# Операторы идентичности
x = [1, 2, 3]
y = [1, 2, 3]
z = x
print(x is y)  # False
print(x is z)  # True
print(x == y)  # True
```
